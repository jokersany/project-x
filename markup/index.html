Selector types

#id - Ідентифікатор визначає унікальне ім'я елемента, яке використовується для зміни його стилю і звернення до нього через скрипти.

.class - Класи застосовують, коли необхідно визначити стиль для одного або декількох елементів веб-сторінки.

Мультикласи - До будь-якого елемента одночасно можна додати кілька класів, перераховуючи їх в атрибуті class через пробіл.

*Універсальний селектор - Іноді потрібно встановити одночасно один стиль для всіх елементів веб-сторінки

Вкладені селектори - Під час створення веб-сторінки часто доводиться вкладати одні елементи всередину інших

Дочірні селектори - Дочірнім називається елемент, який безпосередньо розташовується всередині батьківського елемента.

Сусідні селектори - Сусідні називаються елементи веб-сторінки, коли вони йдуть безпосередньо один за одним у коді документа.

Споріднені селектори - Споріднені селектори за своєю поведінкою схожі на сусідні селектори, але на відміну від них стильові правила застосовуються до всіх прилеглих елементів

Селектори атрибутів - Встановлює стиль для елемента, якщо задано специфічний атрибут. Його значення в цьому випадку не важливе.

атрибут="значення" - Встановлює стиль для елемента в тому випадку, якщо задано певне значення специфічного атрибута.

атрибут^="значення" - Встановлює стиль для елемента в тому випадку, якщо значення атрибута починається із зазначеного тексту.

атрибут$="значення" - Встановлює стиль для елемента в тому випадку, якщо значення атрибута закінчується зазначеним текстом.

атрибут*="значення" - Можливі варіанти, коли стиль слід застосувати до тега з певним атрибутом, коли частиною його значення є деякий текст.
